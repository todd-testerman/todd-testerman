## Featured Projects

### [16S-Workflow](https://github.com/todd-testerman/16S-Workflow)
Pipeline for importing QIIME2 outputs into phyloseq with integrated decontamination (decontam), differential abundance analysis (DESeq2), and visualization (pheatmap).

### [Microbiome_Workflow](https://github.com/todd-testerman/Microbiome_Workflow)
Comprehensive documentation for microbiome data analysis from QIIME2 import through statistical analysis.

### [Code-for-Kravetz-Testerman-2020](https://github.com/todd-testerman/Code-for-Kravetz-Testerman-2020)
Reproducible analysis code supporting research on gut microbiota and PNPLA3 variant effects on NAFLD in obese youth.
